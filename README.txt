What is this?
	A subscription-based web application designed to give users easy access to watch broadcasts online.
Main Features
	Giving users the ability to search for the broadcasts they want to watch,
	easily navigating through channels, as well as choosing their preferred subscription plan.

How to run?
	open terminal
	write this commands:
	1) node -v // this command to check if you have installed node.js
		it should show node version
		if it not the case, install node.js
	2) npm install
	3) npm run dev
